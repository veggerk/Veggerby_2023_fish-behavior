# grad_thesis_ch1

## Shellfish aquaculture farms as foraging habitat for nearshore fishes and crabs


This is the repo for my chapter 1 masters thesis

This repository has the following structure: 

`\R` folder containing R code

`\data\raw_data` folder with raw data files

`\data\clean_data` folder with cleaned data files

`\data\clean\map_files` folder with files to generate the figure 1 map

`\figures` folder with figures

metadata explaining the column names and data source is located in: `metadata.xlsx`
